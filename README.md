# Whale Wave Bill Service

## Usage

### Install

```bash
yarn

yarn start:dev
```

### Nest Cli

create module

```bash
yarn n:g:res <module_name>
```

### Migration

```bash
yarn migration
```

### Swagger

[json同步地址](http://localhost:3001/doc-json)

## Recommended version

> NodeJs version 18.19.0
> Yarn version 1.19.2
